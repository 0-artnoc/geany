Binary Packages

Fedora

You can use the Fedora Core 4 repository from http://naturidentisch.de/packages/fc4/.

Debian

Binary packages for Debian are available at http://debian.uvena.de/. You can add one of the following lines(depending on your system) to your apt sources.list to automatically get the latest version of Geany:

deb http://debian.uvena.de/ ./stable/

deb http://debian.uvena.de/ ./testing/

SuSE

Packages for SuSE are not yet available.

Gentoo

An ebuild for Gentoo can be found on http://www.gentoo.de.
